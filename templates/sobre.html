Sobre o SYSAD-AI

Auxiliando médicos no diagnóstico de doenças cardiovasculares com inteligência artificial

Visão Geral

Nosso sistema utiliza inteligência artificial para oferecer suporte a médicos no diagnóstico de doenças cardiovasculares, ajudando a melhorar a precisão e a eficiência dos diagnósticos.

O que Fazemos

O sistema oferece diagnósticos automáticos baseados em dados clínicos, visualização de histórico de pacientes e geração de relatórios detalhados para análise médica.

Funcionalidades Principais

Diagnóstico Automático

Utiliza algoritmos de aprendizado de máquina para fornecer diagnósticos precisos com base em dados clínicos dos pacientes.

Visualização de Histórico

Permite visualizar e analisar o histórico dos pacientes de maneira intuitiva e fácil de entender.

Geração de Relatórios

Gera relatórios detalhados com informações relevantes para a tomada de decisão médica.

Tecnologias Utilizadas

Flask

Framework web para desenvolvimento rápido de aplicações Python.

scikit-learn

Biblioteca de aprendizado de máquina em Python para construção e avaliação de modelos de IA.

SQLite

Sistema de gerenciamento de banco de dados relacional para armazenamento de dados clínicos.
